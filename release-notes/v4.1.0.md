## v4.1.0

This release improves reliability when requesting structured JSON from LLMs and refines the package’s release-notes generation pipeline to produce cleaner, more compact output.

### Highlights

- Cleaner release notes output: tighter prompts, deduplication/sanitization of generated notes, and more compact evidence referencing.
- More resilient structured-JSON parsing: automatic retries (up to 3 attempts) with an increasing token budget when responses appear truncated.

### Fixes

- Reduced failures caused by partially returned JSON by retrying when the response looks truncated.

### Performance

- Improved the release-notes generation flow to produce more concise, higher-signal notes with less clutter.
