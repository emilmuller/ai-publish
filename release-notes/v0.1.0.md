# Initial Release

This is the first public release of **ai-publish**, providing both a CLI and a programmatic Node.js API.

## Features

- **CLI tool for publishing workflows**
  - Installable as a global or local binary (`ai-publish`) on Node.js 18+.
  - Ships as a CommonJS CLI that integrates with the library to automate release-related tasks.

- **Programmatic Node.js API**
  - Public entrypoint exports a CommonJS library surface that you can import from your Node.js projects.
  - Access building blocks for diff inspection, changelog and release-notes generation, version bumping, and more, all from code.

- **Automated changelog and release-notes generation**
  - Utilities to analyze changes, classify them, and generate structured changelogs.
  - Helpers to produce human-readable release notes based on repository history and diffs.

- **Semantic versioning and multi-language manifest support**
  - Version bump computation helpers and version-base resolution utilities.
  - Support for updating manifests for multiple ecosystems (including npm, Python, .NET, Go, and Rust) as part of your release process.

- **Git- and diff-aware workflows**
  - Integrates with Git to inspect repository history and current state.
  - Diff utilities to index and summarize changes, enabling precise release analysis.

- **LLM integrations**
  - Built-in support for OpenAI and Azure OpenAI clients.
  - Configuration and helper utilities for deterministic behavior and evaluation-oriented workflows.

- **Pipelines for end-to-end automation**
  - High-level pipeline runners for generating changelogs, release notes, and applying version bumps.
  - Pre- and post-publish pipeline hooks so you can automate steps before and after publishing.

- **Editor and tooling support**
  - TypeScript-first codebase with strict typing and emitted type declarations for library consumers.
  - VS Code task integration for building the project.
  - Vitest-based test suite covering CLI, programmatic APIs, pipelines, and manifest behavior.
